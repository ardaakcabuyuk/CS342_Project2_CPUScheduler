Compile the files by typing 'make' on the command line.

Run the executable according to either of these commands
./schedule <N> <Bcount> <minB> <avgB> <minA> <avgA> <ALG>
./schedule <N> <ALG> -f <infile>

For example, if <infile> is “afile”, then the name of the input file for W thread 3 should be “afile-3.txt"
